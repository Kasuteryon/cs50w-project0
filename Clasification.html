¿Qué tipos hay?

Nativas

Son aquellas aplicaciones que se concentran en la utilizacion de los recursos nativos del dispositivo móvil tales como juegos, aplicaciones del sistema a través de las APIs nativas del SO como el uso de la cámara, sensores como el giroscopio, el almacenamiento interno del dispositivo, etc.

Las tecnologías para el desarrollo nativo estan orientadas exclusivamente a un sistema operativo en específico.

Hibridas

Son aquellas aplicaciones que están desarrolladas con una sola base de código la cual es funcional para diferentes plataformas.

Su nombre alternativo es aplicaciones Multiplataforma y como se destaca, son capaces de ejecutarse en mas de 2 sistemas operativos móviles a través del mismo lenguaje. Con la posibilidad de manipular funciones nativas o consumir contenido web.

Compiladas

Funcionan de manera similar a las aplicaciones híbridas, con la diferencia que se compilan al lenguaje nativo de cada una de las plataformas soportadas.

Este tipo de aplicaciones móviles Multiplataforma funciona como una gran alternativa que por su increible flexibilidad como las híbridas pero con un rendimiento cercano a las nativas.

Web

Son aquellas aplicaciones que utilizan al mínimo los recursos nativos del dispositivo pues sus servicios principales están basados en Internet, pues de ahí se consume el contenido principal del app y de manera local solo se almacenan permisos del uso de internet, al igual que el almacenamiento de tokens de identificación y archivos opcionales.

Usualmente son programadas con tecnologías híbridas, pero también pueden serlo con nativas o compiladas.

Nativas


Ventajas
  • Gran Rendimiento
  • Experiencia de Usuario mas fluida
  • Permiten Notificaciones Push
Desventajas
  • Coste de inversión elevado
  • Está orientado solo a una plataforma

Web


Ventajas
  • Opción Económica
  • Costo de Producción reducido
  • Sencillas
Desventajas
  • Pobre utilizacion de componentes nativos
  • Rendimiento sujeto a calidad de conexión del cliente

Híbridas


Ventajas
  • Mantenimiento de baja complejidad
  • El sistema operativo no importa
  • Podés utilizar algunas funciones nativas
Desventajas
  • Funciones Limitadas
  • Rendimiento menor a las nativas

Compiladas


Ventajas
  • Rendimiento Nativo
  • Flexibilidad Multiplataforma
Desventajas
  • Relativamente nuevo
  • Archivos de código base bastante voluminosos
